title: Agent Auth API
---

## Purpose

The **Agent Auth API** is a server that manages the API keys used by agents to authenticate themselves. Using this API, admins can:

- Add a new API key
- Delete an API key
- List all API keys
- View an API key
- Update an API key
- Enable an API key
- Disable an API key
- Revoke an API key
- Rotate an API key

## Authentication

Only admins can access the Agent Auth API.
