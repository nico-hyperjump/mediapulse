title: Dependencies & Prerequisites
---

### External Services

- OpenAI API (GPT-4 access)
- PostgreSQL database (managed or self-hosted)
- Redis (for BullMQ)
- Email service (Resend or SendGrid)
- Market data APIs (Alpha Vantage, Polygon.io)
- SEC EDGAR API (free, public)
- News APIs (NewsAPI, Google News)
- Social media APIs (Twitter/X, Reddit)

### Team Requirements

- Full-stack developers (Next.js, TypeScript)
- Backend engineers (Node.js, queue systems)
- AI/ML engineers (prompt engineering, optimization)
- DevOps engineer (infrastructure, monitoring)
- QA engineer (testing, quality assurance)
- Product manager (coordination, requirements)
